# genetic-algorithm-TSP
Implementation of Genetic Algorithm for TSP in C++
## algorithm
In crossover phase, it uses two operators, Edge Recombination and Order Crossover.
## usage
Input file example (see bays29.txt):
```
1 1150.0 1760.0
2 630.0 1660.0
3 40.0 2090.0
4 750.0 1100.0
5 750.0 2030.0
.
.
.
```
